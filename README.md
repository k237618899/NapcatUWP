# NapCat UWP
-------------

NapCatUWP是一個基於OneBotAPI與NapCat通信實現的UWP版QQ

本項目基於UWP(Universal Windows Platform)，完全由C#實現，
支援**Windows 10 1703**版本以上以及**Windows Phone**的**Windows 10 Mobile 1709**版本安裝使用

由於是個人興趣所至開發的項目，目前實現的東西只有基本的發送文字消息和接收文字/圖片/表情
